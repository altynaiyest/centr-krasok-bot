# 🎨 AI Telegram-ассистент «Центр Красок #1»

Telegram-бот с искусственным интеллектом на базе **Google Gemini 2.0 Flash**. Отвечает на вопросы о компании, используя структурированную базу знаний. Поддерживает контекст диалога (последние 10 сообщений), запрещает галлюцинации, работает на русском, казахском и английском языках.

## 🚀 Функциональность

- ✅ Ответы только на основе предоставленной базы знаний (сайт, соцсети)
- ✅ Поддержка многозначного диалога (история сообщений)
- ✅ Защита от выдумывания фактов (system instruction + fallback)
- ✅ Дружелюбный интерфейс, реакция на любой текст (без команд)
- ✅ Обработка ошибок и вежливый отказ при отсутствии информации
